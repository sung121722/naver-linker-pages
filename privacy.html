개인정보처리방침

최종 업데이트: 2026년 6월

네이버 내부링크 도우미(이하 "서비스")는 사용자의 개인정보를 중요하게 생각하며, 아래와 같이 처리합니다.

1. 수집하는 정보

  • 익명 세션 ID (브라우저 로컬 스토리지에 저장, 서버 전송)
  • 사용자가 입력한 네이버 블로그 ID
  • 검색 키워드 (내부링크 추천 및 중복 감지 기능 수행 목적)
  • IP 주소 (무료 플랜 일일 사용 횟수 제한 목적)

2. 수집 목적

  • 내부링크 추천 및 중복 글 감지 서비스 제공
  • 플랜별 사용 횟수 관리
  • 서비스 품질 개선

3. 보관 및 삭제

수집된 정보는 서비스 제공 목적 달성 후 파기합니다. 블로그 글 목록(제목·URL)은 사용자가 직접 등록을 취소하거나 계정 삭제 요청 시 삭제됩니다.

4. 제3자 제공

수집된 개인정보는 원칙적으로 제3자에게 제공하지 않습니다. 단, AI 내부링크 분석을 위해 입력된 키워드는 Anthropic API(Claude)에 전달됩니다. Anthropic의 데이터 처리 방침은 anthropic.com/privacy에서 확인하실 수 있습니다.

5. 결제 정보

결제는 토스페이먼츠를 통해 처리되며, 카드 정보는 당사 서버에 저장되지 않습니다. 결제 정보 처리 방침은 토스페이먼츠 약관을 따릅니다.

6. 사용자 권리

사용자는 언제든지 수집된 개인정보의 열람, 수정, 삭제를 요청할 수 있습니다. 아래 이메일로 문의해 주세요.

7. 크롬 확장 프로그램 권한 사용 목적

  • storage: 세션 ID 및 블로그 정보 로컬 저장
  • sidePanel: 사이드 패널 UI 표시
  • tabs: 결제 페이지 새 탭 열기
  • scripting: 네이버 에디터에 링크 자동 삽입
  • host_permissions (blog.naver.com): 블로그 글 목록 수집 및 에디터 접근
