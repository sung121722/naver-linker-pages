Features

SEO를 높이는 3가지 핵심 기능

🔗

AI 내부링크 추천

새 글의 주제를 입력하면 AI가 기존 글 전체를 분석해 관련도 높은 글을 최대 50개까지 추천합니다. 내부링크 SEO를 손쉽게 강화하세요.

🔄

중복 글 감지

이미 비슷한 글을 쓴 적이 있는지 사전에 확인합니다. 유사도 점수와 함께 기존 글 목록을 보여줘 콘텐츠 분산을 방지합니다.

⚡

에디터 자동 감지

네이버 블로그 에디터에서 제목을 입력하면 검색어가 자동으로 채워집니다. 검색 버튼 한 번만 누르면 관련 글이 바로 나옵니다.

📊

관련순 · 최신순 정렬

AI 관련도 기준 정렬과 최신 작성일 기준 정렬을 버튼 하나로 전환합니다. 상황에 맞는 글을 빠르게 찾을 수 있습니다.

🏢

멀티 블로그 관리 (Pro)

최대 3개 네이버 블로그를 하나의 계정으로 관리합니다. 블로그 대행 업무에 최적화된 Pro 플랜 전용 기능입니다.

🔒

브라우저 직접 수집

블로그 글 목록은 내 브라우저에서 직접 수집합니다. 네이버 IP 차단 없이 안정적으로 동작하며, 글 본문은 서버로 전송되거나 저장되지 않습니다(제목·주소만 검색에 사용).

How it works

3단계로 바로 시작

1

확장 프로그램 설치

크롬 웹스토어에서 nlinker를 설치합니다. 회원가입 없이 바로 사용 가능합니다.

2

내 블로그 등록

사이드패널에 블로그 ID를 입력하고 수집 버튼을 누르면 글 목록을 가져옵니다. (최초 1회)

3

글 쓰면서 바로 검색

제목을 입력하면 검색어가 자동으로 채워지고, 검색 버튼만 누르면 관련 글을 바로 확인합니다.
